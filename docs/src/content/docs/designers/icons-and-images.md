---
title: Icons and Images
description: Prepare image components for reliable extraction and optional sprite generation.
---

Image tokens are extracted from components in a source frame or component set.

## SVG icons

1. Keep icons as components in a dedicated source area.
2. Keep dimensions consistent for icons in the same family.
3. Separate single-color and multicolor icon sets when possible.

This allows developers to configure output and sprite behavior per icon group.

## Raster assets

1. Store exportable image components in their own frame or set.
2. Name assets for code usage, not presentation-only labels.

## Sprite readiness checklist

1. Icon names are unique.
2. Icons are in a dedicated group.
3. Naming style is consistent across all icons.

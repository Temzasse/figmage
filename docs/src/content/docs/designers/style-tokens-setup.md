---
title: Style Tokens Setup
description: Configure Figma styles that map directly to core Figmage token types.
---

Figmage reads these style systems directly:

1. Color styles -> `color` tokens
2. Text styles -> `text` tokens
3. Drop shadow effects -> `dropShadow` tokens

## Recommended setup

1. Use clear, stable style names.
2. Group related styles with folder naming.
3. Avoid temporary naming like `New Color`.

## Grouping behavior

Top-level grouping in names and folders should reflect how you want tokens grouped in output.

Example groups:

1. `Light/...`
2. `Dark/...`
3. `Web/...`
4. `Native/...`

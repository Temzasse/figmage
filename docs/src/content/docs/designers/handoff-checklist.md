---
title: Handoff Checklist
description: Final checklist before developers run figmage sync.
---

Use this list before each handoff:

1. Access token flow is agreed and secure.
2. File ID is confirmed.
3. Color styles are final and named consistently.
4. Text styles are final and grouped by platform/context.
5. Drop shadows are represented as drop-shadow effects.
6. Property source component sets are published and stable.
7. Icon/image source frames are complete and naming is consistent.
8. Pending renames or structural changes are communicated to developers.
